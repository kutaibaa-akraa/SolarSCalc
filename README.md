 [حاسبة المجموعة الشمسية - محاكاة ثنائية وثلاثية الأبعاد] (https://kutaibaa-akraa.github.io/SolarSCalc/solar_system.html) 

**Solar System Calculator - 2D & 3D Simulation**

---

### نبذة عن المشروع

هذه أداة تفاعلية متقدمة لحساب مواقع الكواكب والأجرام السماوية في المجموعة الشمسية، مع إمكانية عرضها في **جدول بيانات رقمي**، **رسم ثنائي الأبعاد (Canvas)**، و **فضاء ثلاثي الأبعاد (Three.js)**.

تم تطوير هذه الواجهة بالاعتماد على محرك الحسابات الفلكية  لـ [Don Cross](http://www.cosinekitty.com)، مع ترجمة شاملة للواجهة إلى اللغة العربية وتطويرات هيكلية وبصرية كبيرة لتوفير تجربة مستخدم احترافية وسهلة.

### الميزات الرئيسية

| الميزة | الوصف |
| :--- | :--- |
| ** الحسابات الفلكية الدقيقة** | يعتمد على معادلات Paul Schlyter لحساب مواقع الكواكب، الكويكبات، والمذنبات بدقة تصل إلى بضع دقائق قوسية. |
| ** جدول إحداثيات تفاعلي** | عرض الإحداثيات المتعامدة (X, Y, Z) بوحدة AU، القدر الظاهري، الاستطالة، الكوكبة، والإحداثيات الزاوية (استوائية، بروجية، سمتية). |
| **️ رسم ثنائي الأبعاد (2D)** | منظر علوي من القطب الشمالي للمجموعة الشمسية مع شبكة إحداثية مرقمة، دوائر مدارية، وتكبير/تصغير باستخدام عجلة الفأرة. |
| ** رسم ثلاثي الأبعاد (3D)** | فضاء تفاعلي باستخدام Three.js يتيح السحب والتدوير والتكبير لاستكشاف الكواكب من أي زاوية، مع تسميات للأجرام ومحاور إحداثية واضحة. |
| **️ خريطة تفاعلية** | اختيار موقع الراصد جغرافياً عبر خريطة Leaflet (بالنقر أو سحب العلامة)، مما يؤثر على أوقات الشروق والغروب والإحداثيات السمتية. |
| **️ محاكاة الزمن المتقدمة** | التحكم بسرعة الزمن (إبطاء، تسريع، مضاعفة) حتى 10000x، إيقاف مؤقت، والعودة الفورية إلى الوقت الحقيقي، لمشاهدة حركة الكواكب عبر القرون. |
| ** واجهة عربية حديثة** | تصميم داكن (Dark Mode) مع ألوان بيج دافئة ومتغيرات CSS لتوحيد المظهر، مع دعم كامل للغة العربية. |
| ** حفظ التفضيلات** | استخدام الكوكيز لحفظ إعدادات المستخدم (نظام الإحداثيات، الموقع الجغرافي، سرعة المحاكاة، وحالة الفلاتر) لاستمرار التجربة بين الجلسات. |
| ** فلاتر العرض** | إظهار الأجرام اللامعة فقط، الأجرام فوق الأفق فقط، وإظهار/إخفاء المذنبات والكويكبات الصغيرة. |

### التقنيات المستخدمة

- **HTML5 / CSS3** – بناء الواجهة وتنسيقها (مع متغيرات CSS المخصصة).
- **JavaScript (ES6)** – منطق الحسابات والتفاعل.
- **Three.js** – العرض ثلاثي الأبعاد (مع OrbitControls للتحكم بالكاميرا).
- **Leaflet.js** – الخريطة التفاعلية لتحديد الموقع الجغرافي.
- **Font Awesome** – أيقونات التحكم (رمادية احترافية).

### كيفية التشغيل (محلياً)

1.  **قم بتحميل أو استنساخ المستودع**:
    ```bash
    git clone https://github.com/kutaibaa-akraa/SolarSCalc.git
    ```
2.  **افتح الملف الرئيسي**:
    - انتقل إلى مجلد المشروع.
    - افتح ملف `solar_system.html` (أو `index.html`) باستخدام متصفح ويب حديث (Chrome, Firefox, Edge).
    - **ملاحظة**: لا حاجة لخادم ويب، حيث أن جميع المكتبات يتم تحميلها عبر CDN، والمشروع يعمل كصفحة ثابتة (Static Page).

### دليل استخدام سريع

1.  **اختيار نظام الإحداثيات**: استخدم أزرار الراديو لتحديد "مركزه الشمس" أو "مركزه الأرض" لمشاهدة الإحداثيات المتعامدة.
2.  **استكشاف الجدول**: اقرأ قيم `X, Y, Z`، القدر الظاهري، والزوايا. مرر الماوس فوق الصفوف لتمييزها.
3.  **التحكم بالزمن**:
    - فعّل "الوقت الحقيقي" للتحديث التلقائي.
    - عطّل الوقت الحقيقي وعدّل التاريخ/الوقت يدوياً، ثم اضغط "تعيين".
    - استخدم أزرار السرعة (`10x`, `100x`, ...) لتسريع المحاكاة ومشاهدة حركة الكواكب.
4.  **التنقل بين العروض**:
    - شاهد الرسم ثنائي الأبعاد على اليسار.
    - اضغط زر **"🌍 عرض 3D"** للتبديل إلى الفضاء ثلاثي الأبعاد، واسحب بالماوس لتدويره.
5.  **تحديد الموقع**: انقر على الخريطة أو اسحب العلامة لتحديد موقعك الجغرافي، مما يغير حسابات الارتفاع والسمت.
